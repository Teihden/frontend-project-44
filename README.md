## Setup

```shell
# Install dependencies
make install
```

## Run

```shell
# Run package
make brain-games
```

## CLI

```shell
# Runs npm-publish with flag (--dry-run)
make publish
```

```shell
# Runs Eslint against source code for quality
make lint
```
